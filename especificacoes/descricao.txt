
Nesse diretório você vai encontrar vários arquivos, necessários para desenvolver o trabalho do CESAR

Os primeiros arquivos dizem respeito à especificação do trabalho propriamente dito.
São os seguintes:
	- Apresentacao2023_1.pdf - descrição geral sobre o trabalho.
	- TrabalhoCesar2023_1.pdf - detalhamento do trabalho, que envolve o desenvolvimento de um kernel.
	- FuncoesDoKernel2023_1.pdf - detalhamento das funções do kernel a ser implementado.
	- kernel_ref.ced - arquivo fonte para ser usado como ponto de partida para a implementação.

Os outros arquivos são fornecidos para facilitar o teste de seu kernel.
São cinco programas de aplicação, que devem ser montados e carregados na área de memória reservada para as aplicações.
Cada aplicação utiliza algumas das funções que o kernel deve implementar.
Verifique quais são as funções no próprio fonte dos programas.
Caso necessário, você pode alterar esse programas conforme sua necessidade.
	- app*.ced - aplicações de teste, conforme descrição.
